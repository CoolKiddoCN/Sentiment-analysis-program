This compressed package contains the original raw corpus file (raw corpus.xlsx), the corpus preprocessing script (preprocessing.py), the cleaned and tokenized corpus (cleaned corpus.txt), the sentiment analysis script (Sentiment analysis.py), and the sentiment analysis results (Sentiment analysis result.xlsx).
Detailed information regarding the code execution environment and usage can be found in Readme.md.
